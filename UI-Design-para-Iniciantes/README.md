# UI Design para Iniciantes

Curso sobre Design de Interface de Usuário (UI Design) utilizando Figma como ferramenta para criação dos protótipos.

O curso aborda conceitos como: formas, organização do projeto, imagens e otimizações, espaçamento, tipografia, paleta de cores, luz e sombra e design responsivo.
